==  WHAT'S INCLUDED IN OUR FREE MWB GF INTEGRATION WITH ZOHO CRM PLUGIN ==

**1) Accounts to Connect**
Enter your ZOHO API Key to connect your ZOHO account to Gravity Forms.

**2) Mapping of Fields**
Gravity Forms entry fields could be considered equivalent to any ZOHO Object attributes.

**To Update an Existent Entry in ZOHO CRM, use the Primary Key**
Assume that a user entry from a Gravity Form submission already exists in ZOHO CRM. In that instance, the administrator can update it using the "Primary Key" provided. 
In this approach, integrating Gravity Forms ZOHO CRM Campaigns becomes a breeze.

**3) Numerous Feeds**
A feed controls how data input from Gravity Forms should be transmitted to ZOHO CRM. Multiple feeds can be created and conditions for relaying ZOHO CRM entries setup.

**4) Reporting Errors**
If an error is encountered while sending the data on ZOHO CRM using Gravity Forms, an e-mail would be sent to the administrator informing about the fault description.

**With any ZOHO module, Feeds creates a field association**
Gravity Forms Integration With ZOHO CRM allows an administrator to link any Gravity Form field to any readily accessible ZOHO module field using feeds. Any module, including Contacts, Accounts, Leads, Quotes, Deals, Sales, and Orders, could be used with this plugin.

**5) Logs for CRM**
View a detailed log of every Gravity Forms entry sent to ZOHO CRM.

**Export Logs as CSV**
Easily export ZOHO CRM Logs as a CSV file for forecasting, book-keeping, and accounting purposes.

**6) Synchronization**
All Gravity Forms entries are fully synchronized with ZOHO CRM. If you create/update an entry that entry will be created/updated in ZOHO CRM.

**7) Filter Entries**
By default, all Gravity Forms entries are sent to ZOHO CRM, but you can apply filters & set up rules to limit the entries sent to ZOHO CRM.

== Frequently Asked Questions ==

= What is the basic purpose of using the MWB GF Integration with ZOHO CRM plugin? =

The MWB GF Integration with ZOHO CRM plugin sends all Gravity Form data submissions to ZOHO CRM as per the form field mappings and classifies them on the basis of various objects and ZOHO modules assigned. 

= How to Authorize the Connection of Your ZOHO Accounts? =

The admin needs to have the ZOHO account or can directly sign up using the link we have provided in our plugin in the ZOHO Account tab and then proceed with selecting the Zoho Domain. Then enter the Client ID, Secret Key & Redirect URL after generating it following the instructions shared in a plugin. The instructions can be seen by clicking the plus icon in the top right corner.

= How would I use the MWB GF Integration plugin to synchronize ZOHO CRM with Gravity Forms? =

By entering your ZOHO API credentials, the Gravity Form can be integrated with your ZOHO account.
Enabling the MWB GF Integration with the ZOHO CRM plugin adds a ZOHO sub-menu to the Gravity Form. 
The administrator can easily sign up for or log into his ZOHO CRM account and connect Gravity Forms to his ZOHO CRM account.

=  Is there an option to add new feeds using MWB GF Integration with ZOHO CRM? =

Yes, utilizing MWB GF Integration with ZOHO CRM, really is possible.

The admin can map any Gravity Form field to any available ZOHO module field using the MWB GF Integration with ZOHO CRM. This plugin works with any module, such as
Contacts, Accounts, Leads, Quotes, Deals, and Sales.
The administrator is free to add as many feeds as he prefers, just require the specified Gravity Form with the ZOHO Object. 

In order to create feeds, you need to traverse within your WordPressDashboard > Form > ZOHO > ZOHO Feeds.
They either make use of any already created feed or create one by using the Add New Feed button.
